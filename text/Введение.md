Моноклональные антитела (mAbs) являются одним из быстрорастущих классов терапевтических препаратов. Традиционный процесс разработки включает иммунизацию животных, гибридомную технологию или фаговый дисплей с последующим скринингом кандидатов — процесс, занимающий месяцы и требующий значительных ресурсов. За последние годы появился ряд генеративных языковых моделей для дизайна антител (IgLM, AbLang, p-IgGen), однако большинство из них являются antigen-agnostic — генерируют антитела без учёта информации о целевом антигене. Существующие antigen-aware подходы либо требуют структурной информации об антигене, либо работают только с одной цепью антитела. Модель p-IgGen представляет интерес как основа для antigen-conditioned генерации, поскольку генерирует парные последовательности. Появление базы данных AgAbDb, содержащей более миллиона пар антиген–антитело с данными об аффинности, открывает возможность для дообучения языковых моделей в antigen-conditioned режиме. В связи с этим актуальна разработка методов de novo генерации антител с учётом последовательности целевого антигена.
